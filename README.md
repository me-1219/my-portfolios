# Mebit Melak — Portfolio (React + Vite)

Live CV: https://cv.melaklegacy.com.et

A responsive personal portfolio built with React, Vite and TailwindCSS. Includes a hero, about, projects, certifications, testimonials, contact form and downloadable CV.

---

## Features

- Clean, responsive UI (mobile → desktop)
- Dark / Light theme with localStorage persistence
- Animated hero (Framer Motion)
- Projects grid with images imported from `src/assets`
- Certifications gallery (public or imported assets)
- Downloadable CV (PDF) and social links
- Smooth scrolling to page sections

---

## Tech stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- react-icons
- (Optional) Netlify / Vercel for deployment

---
